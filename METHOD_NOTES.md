# 方法笔记

## 算法总体流程

```
输入字形图像
    ↓
图像预处理（灰度化、二值化、去噪）
    ↓
骨架提取（Zhang-Suen / 形态学细化）
    ↓
骨架点排序 → 初始轨迹点序列
    ↓
强化学习局部优化（DDPG）
    ↓
优化后的机器人末端轨迹
```

## 1. 图像预处理

- 灰度化：将RGB图转为灰度图
- 二值化：Otsu自适应阈值
- 去噪：中值滤波 / 高斯滤波

## 2. 骨架提取

- 算法：Zhang-Suen 细化算法
- 输出：单像素宽度的字形骨架
- 骨架点排序：从起笔位置开始，沿骨架路径排序

## 3. 初始轨迹生成

- 从骨架点生成机器人末端连续轨迹
- 可选的B样条平滑处理

## 4. 强化学习优化（DDPG）

### 状态空间
- 当前轨迹点的位置 (x, y)
- 局部曲率信息

### 动作空间
- 轨迹点的微调偏移量 (Δx, Δy)

### 奖励函数
- 形状保真度项：优化后轨迹与目标骨架的匹配度
- 平滑性项：相邻轨迹点间的曲率惩罚
- R = α × shape_similarity + β × smoothness

### 网络结构
- Actor：MLP，输入状态 → 输出动作
- Critic：MLP，输入状态+动作 → 输出Q值

## 备选方案（如RL训练不稳定）

- 使用"强化学习思想 + 局部搜索/策略迭代"
- 使用改进Q-learning替代DDPG
- 使用PPO替代DDPG
