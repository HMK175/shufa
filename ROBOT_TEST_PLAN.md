# 机器人测试计划

## 实验条件说明

- 可用设备：机械臂（末端夹爪夹持书写笔）
- 不做毛笔书法（不需要笔压、笔锋、墨迹扩散）
- 重点验证：**轨迹规划是否合理**，而非真正毛笔书法艺术效果

## 测试目标

验证算法生成的轨迹能够被机械臂执行，并完成简单字形书写。

## 测试流程

### 第一阶段：纯仿真验证
1. 输入黑白字形图片
2. 运行算法生成优化轨迹
3. 在二维平面绘制轨迹并与目标字形对比
4. 计算定量指标（RMSE、平滑度等）

### 第二阶段：运动学映射
1. 将二维轨迹点映射到机械臂工作空间坐标系
2. 进行运动学逆解，生成关节角度序列
3. 在仿真环境（如ROS/MATLAB）中验证机械臂运动范围

### 第三阶段：实物书写实验（如有条件）
1. 3-5条已平滑的轨迹，不现场生成
2. 机械臂末端夹爪夹持书写笔
3. 执行轨迹跟踪
4. 拍摄书写结果并与目标字形对比

## 论文中描述方式

- 不强调"毛笔书法效果"，而说"末端轨迹跟踪实验"
- 实验采用仿真与实物验证相结合的方式
- 首先在二维平面中验证轨迹生成与优化效果
- 随后将优化轨迹映射到机械臂工作空间，通过末端夹爪夹持书写笔完成简单字形轨迹跟踪实验

## 最低验证标准

1. 能输入一张字形图片 ✓
2. 能输出一条或多条轨迹 ✓
3. 优化后轨迹误差比优化前低 ✓
4. 有5个以上样本验证 ✓
5. 有至少2个对比方法 ✓
6. 能映射到机械臂工作空间 ✓
7. 有仿真图或实物图 ✓
8. 论文能解释为什么有效 ✓
