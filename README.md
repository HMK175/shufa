# 基于图像骨架提取与强化学习优化的书法机器人轨迹生成方法

## 项目简介

本项目针对书法机器人书写轨迹人工示教成本高、轨迹还原度不足的问题，提出一种结合图像骨架提取和强化学习局部优化的笔画轨迹生成方法，实现了由目标字形图像到机器人末端轨迹的自动生成与优化。

**核心流程**：输入字形图像 → 图像预处理 → 骨架提取 → 初始轨迹生成 → 强化学习优化 → 机器人末端轨迹输出

## 小论文定位

- 不做完整汉字生成，不做复杂大模型
- 只做"给定笔画/字形图像后，机器人如何写得更像"这一小问题
- 面向工程实现的轻量化方案：骨架提取 + RL局部优化
- 适用于中文普刊、专硕毕业要求
