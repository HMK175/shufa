# Intelligent Chinese Calligraphy Beautification from Handwritten Characters

- **发表信息**: Xinyue Zhang et al. (南京信息工程大学/岩手大学), The Visual Computer 35, pp.1193-1205, 2019
- **相关度**: ★★
- **方法**: 骨架笔画识别(CNN) + 全局优化匹配 + 标准笔画库
- **流程**:
  1. 用户手写骨架字符（鼠标/触控板）
  2. CNN 识别每个骨架笔画
  3. 全局优化：最小化手写笔画 ↔ 标准书法笔画库的差异
  4. Shape Character Vector (SCV) 描述笔画形状
  5. 机器人书写美化后的笔画

- **主要贡献**:
  - 保留用户风格的同时美化书写
  - SCV 形状描述向量用于笔画匹配优化
  - 端到端系统：手写 → 美化 → 物理书写

- **不足/局限**:
  - 依赖预建的标准笔画库
  - 仅处理楷体
  - 优化仅调整笔画形状参数，不涉及轨迹平滑

- **与本课题的关系**:
  - SCV 形状描述向量可参考用于轨迹评价
  - 但总体思路（标准笔画库匹配）与本课题（RL 优化）不同
