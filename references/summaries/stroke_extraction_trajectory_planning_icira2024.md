# Application of Stroke Extraction and Trajectory Planning in Robotic Calligraphy

- **发表信息**: Yuzhi Wu, Jiashu Feng, Wangcheng Chen, Yisheng Guan (广东工业大学 BIRL), ICIRA 2024, LNAI 15202, pp. 308–320, 2025
- **方法**: 传统图像处理（角点检测+连通域）+ CNN 笔画识别 + 骨架特征点分类的动态路径规划
- **流程**:
  1. 输入文字图像（相机拍摄字帖或现场书写）
  2. 字符分割：角点检测 → 角点匹配与连线（4条约束规则）→ BGR 通道分离 → 连通域提取 → 相似度去重
  3. CNN 笔画识别（3层5x5卷积 + 2x2池化 + dropout + FC64 + softmax，28类笔画，数据集每类100张增强后）
  4. 骨架提取（轮廓数据 → 中点骨架）
  5. 轨迹规划：骨架点分类（顶点V/交叉点S/连接点C）→ 根据书法书写习惯确定方向 → 动态生成路径点序列 → 坐标系变换到机械臂
  6. 仿真验证（ROS RVIZ + Unitree Z1 + MoveIt）

- **主要贡献**:
  - 提出角点匹配的4条约束规则（角度条件 + 命题逻辑 + 距离阈值 + 最近距离差），实现高精度字符笔画分割
  - 动态路径规划算法：不同于简单删除骨架小分支，通过特征点分类（用 Nc 替代 Nw 避免误分类）和交叉点属性动态更新，正确处理笔画交叉处的路径规划（含折返路径）

- **不足/局限**:
  - 仅处理规范楷体字，未涉及行书/草书等风格变化
  - 未考虑笔画宽度与笔刷高度的关联（列为 future work）
  - 轨迹规划基于规则而非学习，缺乏对不同风格的适应能力
  - 无强化学习优化环节
  - CNN 笔画分类仅 28 类，800 张原始图片，规模较小

- **与本课题的关系**:
  - 与本课题 pipeline 方向一致（骨架 → 轨迹 → 机器人），可参考其轨迹规划的特征点分类思路
  - 其纯规则式路径规划正是本课题 RL 优化想要改进的方向
  - 笔画分割的角点匹配方法可作为本课题图像预处理的技术储备
