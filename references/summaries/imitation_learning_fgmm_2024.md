# An Image-Based Imitation Learning Framework for Robotic Writing Tasks

- **发表信息**: Wenjin Xu et al. (华南理工/Liverpool), 2024
- **相关度**: ★★★
- **方法**: 模糊高斯混合模型 (FGMM) + 动态运动基元 (DMP) 的模仿学习
- **流程**:
  1. 静态图像 → FGMM 拟合笔画轨迹（PCA 降维 + 抛物线投影 + 弧长参数化）
  2. 偏差分析方法提取笔画粗细信息
  3. DMP 泛化到不同尺寸的纸张

- **主要贡献**:
  - FGMM 将静态图像转为动态轨迹（零示教数据）
  - DMP 实现技能泛化（不同纸张尺寸）
  - 笔画粗细的偏差分析方法

- **不足/局限**:
  - FGMM 对复杂笔画（交叉/重叠）拟合效果存疑
  - DMP 泛化仅限于尺寸缩放，不涉及风格
  - 未涉及 RL 反馈优化

- **与本课题的关系**:
  - FGMM 图像→轨迹的转换与本课题骨架→轨迹类似
  - DMP 泛化思路轻量但不如 RL 灵活
  - 作为可对比的基线方法
