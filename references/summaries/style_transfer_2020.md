# A Robot Calligraphy Writing Method Based on Style Transferring Algorithm

- **发表信息**: Dong-tai Liang et al., Intelligent Service Robotics 13, pp.137-146, 2020
- **相关度**: ★★
- **方法**: GAN 风格迁移 + 图像特征提取 + 相似度评价
- **流程**:
  1. 输入字符图像 → GAN 风格迁移 → 目标风格字符
  2. 图像处理(二值化/去噪/形态学/细化)提取特征参数
  3. 特征参数 → 机器人控制参数
  4. 相似度评价：平衡度、倾斜度、书写力度

- **主要贡献**:
  - GAN 风格迁移应用到机器人书写
  - 多维相似度评价（平衡/倾斜/力度）

- **不足/局限**:
  - 风格迁移效果受限于训练集
  - 图像处理链路过长，误差累积
  - 仅处理单个字符，无笔画级控制

- **与本课题的关系**:
  - 较弱：风格迁移不是本课题重点
  - 相似度评价指标（平衡度/倾斜度）可参考
