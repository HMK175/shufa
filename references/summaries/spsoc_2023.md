# SPSOC: Staged Pseudo-Spectral Optimal Control for Robotic Chinese Calligraphy

- **发表信息**: Dongmei Guo, Huasong Min, Guang Yan (武汉科技大学/安徽理工大学), 2023
- **相关度**: ★★★★
- **方法**: 伪谱最优控制 (Pseudo-Spectral Optimal Control) + 分段轨迹优化
- **流程**:
  1. 骨架提取 → 最小二乘拟合得到最优轨迹曲线
  2. 分阶段(起笔/行笔/收笔)应用伪谱最优控制
  3. 同时优化轨迹和机械臂控制参数
  4. 确保每阶段参数符合书写规则

- **主要贡献**:
  - 将书法书写描述为轨迹优化问题，用伪谱法求解
  - 分阶段优化：不同书写阶段有不同的控制目标
  - 同时考虑路径和控制参数的优化

- **不足/局限**:
  - 伪谱法计算量大
  - 需要手工设计每阶段的代价函数
  - 规则式方法，对不同风格需重新设计参数

- **与本课题的关系**:
  - 分段控制的思路可参考（起笔/行笔/收笔各有不同优化目标）
  - 轨迹优化框架可对比：SPSOC 用传统最优控制，本课题用 RL——各有优劣
